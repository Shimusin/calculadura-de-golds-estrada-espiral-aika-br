# Calculadora de Gold para O AIKA BR 💰

Uma ferramenta simples e leve de código aberto (open-source) feita em Python para ajudar os jogadores do AIKA a calcularem rapidamente o valor total de seus itens na lojinha, já descontando as taxas de venda (0%, 5% ou 15%).

![Print do Programa](https://imgur.com/zVLM96f.png)

## 🌟 Funcionalidades
- **Cálculo Automático de Taxas:** Mostra o lucro real ao vender itens na loja.
- **Itens Pré-Salvos:** Já vem com [Lingote de Heliotropo, Couro Cristalino, etc] configurados.
- **Adição de Itens:** Você pode adicionar seus próprios itens personalizados na segunda aba. A calculadora faz a matemática reversa para descobrir o valor bruto automaticamente!
- **Interface Amigável:** Design limpo, arredondamento de valores e cálculos instantâneos ao apertar "Enter".

## 📥 Como Baixar e Usar

**Opção 1: Arquivo Executável (.exe) - Mais fácil**
1. Vá na aba de [Releases](link_da_aba_releases_aqui) aqui do lado direito.
2. Baixe o arquivo `Calculadora_Gold.exe`.
3. É só abrir e usar! (Não precisa instalar nada).
*Nota: Como é um programa feito de fã para fã e sem um certificado digital pago, o Windows Defender pode dar um aviso de "Software Desconhecido". Basta clicar em "Mais informações" > "Executar assim mesmo".*

**Opção 2: Rodar pelo Código-Fonte (Para quem tem receio ou é programador)**
1. Baixe os arquivos deste repositório.
2. Certifique-se de ter o Python instalado.
3. Abra o terminal na pasta e digite: `python calculadora_gold.py`.

## 🤝 Transparência
Criei este repositório justamente para que a comunidade possa verificar o código-fonte do programa. Ele é seguro, não tem acesso à internet e não afeta os arquivos do jogo.
